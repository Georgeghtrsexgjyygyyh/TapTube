НЕВИДИМЫЙ VPN

XTLS-Reality // Anti-DPI // Roblox Ready

Твой личный сервер, который маскируется под обычные сайты (Microsoft, Google, Apple). Роскомнадзор видит только безобидный HTTPS-трафик. Ты видишь открытый интернет и Roblox.

ВНИМАНИЕ: Это "хитрый" метод маскировки. Ваш трафик будет выглядеть как обычное посещение легальных западных сайтов. Для работы требуется VPS-сервер за границей (~3-5$/мес).
Автоустановка сервера

Одна команда для установки панели 3X-UI с XTLS-Reality. Работает на Ubuntu 20.04/22.04.

root@vps:~#
# Обновление системы
apt update && apt upgrade -y

# Установка 3X-UI (включает Xray + Reality)
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)

# После установки открой в браузере:
http://YOUR_SERVER_IP:2053
Логин: admin | Пароль: admin

Генератор настроек Reality

Настрой "украденный" сайт для маскировки. Рекомендуем использовать крупные корпоративные сайты.

vless:// конфигурация
Настройка в панели 3X-UI
# 1. Войди в панель http://IP:2053
# 2. Inbounds -> Add Inbound
# 3. Protocol: VLESS
# 4. Port: 443
# 5. Security: Reality
# 6. Dest: выбранный домен выше
# 7. Server Names: тот же домен
# 8. Private Key: (сгенерировать в панели)
# 9. Add User -> UUID: (из поля выше)
# 10. Flow: xtls-rprx-vision
Windows / Linux

Nekoray — лучший клиент с поддержкой Reality.

1. Скачай: github.com/MatsuriDayo/nekoray
2. Запусти nekoray.exe
3. Server -> Import -> Вставь ссылку vless://
4. Правой кнопкой -> Start
5. System Proxy -> Enable
Скачать Nekoray
Android

v2rayNG — официальный клиент для Android.

1. Установи v2rayNG из GitHub
2. "+" -> Import from Clipboard
3. Выбери профиль
4. Нажми кнопку "V" для подключения
5. Запусти Roblox
Скачать v2rayNG
iOS (iPhone/iPad)

Streisand — бесплатный клиент.

1. App Store -> Streisand
2. "+" -> Scan QR or Import
3. Вставь ссылку или отсканируй QR
4. Подключись
5. Работает в фоне
* Требуется App Store аккаунт не из РФ
